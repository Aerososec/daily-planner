# daily-planner
Мобильное приложение-ежедневник

## Реализованный функционал

### Экран календаря и списка задач

* Выбор дня с помощью календаря
* Отображение задач в формате почасовой сетки (00:00 – 23:00)
* Автоматическое обновление списка при выборе даты
* Отображение всех задач в соответствующем временном интервале
* Возможность просмотреть подробности о задачи при нажатии на соответствующую ячейку
* Кнопка для создания новой задачи

### Экран создания и редактирования задачи

* Создание новой задачи
* Редактирование существующей задачи
* Выбор даты через DatePicker
* Выбор времени начала и окончания через TimePicker
* Ввод названия и описания задачи
* Навигация между несколькими задачами в одном временном диапазоне
* Удаление задачи
* Проверка данных на корректность:
  * Нельзя создать задачу в прошлом
  * Время окончания должно быть больше времени начала
